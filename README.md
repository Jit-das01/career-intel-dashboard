# Career Intel Dashboard
### AI-powered College Major × Salary Analysis

Does your college major predict your salary? I built an AI to find out.

## 🚀 How to Run Locally

Want to test the predictor yourself? Follow these steps:

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Jit-das01/career-intel-dashboard.git
   cd career-intel-dashboard
   ```

2. **Install dependencies:**
   ```bash
   pip3 install -r requirements.txt
   ```

3. **Run the App:**
   ```bash
   python3 app.py
   ```
